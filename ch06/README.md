# 6.데이터 타입
- 원시타입
  - 숫자(number)타입
  - 문자열(string)타입
  - 불리언(boolean)타입
  - undefined타입
  - null타입
  - 심벌(symbol)타입
- 객체타입
  - 객체,함수,배열 등

## 6.1숫자 타입
- 하나의 숫자 타입만 존재한다.(정수형,실수형 모두 같다. 모든 수를 실수로 처리.)
- NaN(not a number)

## 6.2문자열 타입
- 문자열은 '',"",``으로 텍스트를 감싼다.
- 원시타입이며, 변경 불가능한 값이다.

## 6.3템플릿 리터럴
- 템플릿 리터럴은 런타임에 일반 문자열로 변환되어 처리된다.
- ``을 사용해 표현한다.
- + 연산자는 피연산자 중 하나 이상이 문자열인 경우 문자열 연결 연산자로 동작한다.
